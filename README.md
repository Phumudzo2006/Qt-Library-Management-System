# Qt Library Management System

## 📌 Description
A desktop application built with C++ and Qt to manage library operations such as books, users, and borrowing.

## 🚀 Features
- Add and manage books
- Register users
- Borrow and return books
- Database management

## 🛠️ Technologies Used
- C++
- Qt Framework

## ▶️ How to Run
1. Open the project in Qt Creator
2. Open `LibrarySystem.pro`
3. Build and run the project
